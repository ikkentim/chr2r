Controls

Right arrow - move right
Left arrow - move left
Down arrow - duck
Enter - Pause/unpause, restart game after gameover

Unlockable abilities
B - Sneaking
N - Sprint
S - Splash attack
Space - Jump

Cheats:

Open console with tab, 'commands' to show all commands

startlevel <level file>

lvl/level01.dat
lvl/mario01.dat
